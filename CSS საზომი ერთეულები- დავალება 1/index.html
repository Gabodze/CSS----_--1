CSS ერთეულები

CSS–ს სიგრძის გამოსახატად აქვს რამდენიმე ერთეული. ბევრი თვისება იძენს 'სიგრძის' მნიშვნელობას, მაგალითად პადინდი, მარჯინი, ფონტის ზომა და ა.შ. სიგრძე არის ის რიცხვი, რომელსაც მოჰყვება სიგრძის საზომი ერთეული. რიცხვსა და ერთეულს შორის არ უნდა გვქონდეს დაშორება. ამასთან, თუ მნიშვნელობა არის ნულის ტოლი, ამ შემთხვევაში სიგრძის საზომი ერთეულის დაწერა აღარ გვჭირდება. ზოგიერთ CSS თვისებას შესაძლოა ჰქონდეს უარყოფითი სიგრძეც.


ორი ტიპის სიგრძის საზომი ერთეული გვაქვს: აბსოლუტური და ფარდობითი (რელაციური).

აბსოლუტური სიგრძე არის ფიქსირებული და ნებისმიერი ზომის ეკრანის ზომაზე ის ზომა იქნება, რასაც მივუთითებთ. სწორედ ამიტომ აბსოლუტური სიგრძის ერთეულის გამოყენება არ არის რეკომენდებული ეკრანისთვის, რადგან ეკრანის ზომა მოწყობილობის მიხედვით არის ცვალებადი. თუმცა მისი გამოყენება შეგვიძლია პრინტტან მუშაობისას.

სიგრძის საზომი ერთეულებია:

  • cm = სანტიმეტრი
  • mm = მილიმეტრი
  • in =ინჩი (in = 96px = 2.54cm)
  • px = პიქსელი (1px = 1/96 of in)
  • pt = წერტილი (1pt = 1/72 of in)
  • pc = picas (1pc = 12pt)
* პიქსელი მოწობილობისთვის არის ფარდობითი. იმ მოწყობილობებისთვის, რომელთა ფიზიკური რეზოლუცია, გაფართოება არის 1000 წერტილზე ნაკლები ერთ ინჩში, 1 პიქსელი არის მოწყობილობის ეკრანის ერთი წერტილი. პრინტერებისთვის და მაღალი გაფართოების ეკრანისთვის ერთი პიქსელი გულისხმობს მოწყობილობის რამდენიმე პიქსელს.

ფარდობითი სიგრძე აჩვენებს სხვა სიგრძის საზომ თვისებასთან შეფარდებულ სიგრძეს.

  • em = ელემენტის ფონტის ზომის ფარდობითია. (2em ნიშნავს მოცემული ფონტის ორჯერ დიდ ზომას, ანუ გაამრავლებს ორზე და თუ გვქონდა 24, გვექნება 48)
  • ex = მოცემული ფონტის x-სიმაღლესთან არის ფარდობითი
  • ch = მოცემული ფონტის "0"ის წიგანესთან არის ფარდობითი
  • rem = ძირეული ელემენტის ფონტის ზომასთნან არის ფარდობითი
  • vw = ბრაუზერის ფანჯრის სიგანის 1%–ის შესაბამისია.
  • vh = ბრაუზერის ფანჯრის სიგრძის 1%–ის შესაბამისია.
  • vmin = გამოხატავს 1%–ს ეკრანის ზომის მინიმალურ სიგრძესთან ან სიგანესთან შეფარდებით.
  • vmax = გამოხატავს 1%–ს ეკრანის ზომის მაქსიმალურ სიგრძესთან ან სიგანესთან შეფარდებით.
  • % = მშობელი ელემენტის ფარდობითია.
